# Campus Booking System

A modern campus room booking system built with PHP, MySQL, Docker and DevOps practices.

---

## Features

- Room booking system
- User authentication
- Admin dashboard
- Docker support
- phpMyAdmin integration
- CI/CD Pipeline with GitLab
- PHPUnit testing

---

## Technologies

- PHP 8.2
- MySQL 8
- Docker
- Docker Compose
- GitLab CI/CD
- PHPUnit

---

## Start Project

```bash
docker compose up --build
